Java

MGX requires a working Java Development Kit (JDK) to be installed on the computer. Typically, Java is already installed on most systems or can be obtained free of charge from

http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html.

The installed Java version can be checked e.g. at

http://www.java.com/de/download/installed.jsp.
